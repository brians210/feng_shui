# Feng Shui App

Bazi (Four Pillars of Destiny) calculator and chart viewer built with React + TypeScript.

## Tech Stack

- React 19 + TypeScript 5.9
- Vite 8
- Ant Design 6 (UI components)
- date-fns / date-fns-tz (date handling)
- @aharris02/bazi-calculator-by-alvamind (Bazi calculation engine)

## Commands

- `npm run dev` — start dev server
- `npm run build` — typecheck + production build
- `npm run lint` — ESLint
- `npm run preview` — preview production build

## Project Structure

- `src/components/` — React components (InputForm, BaziChart, LuckPillars, etc.)
- `src/App.tsx` — root app component
- `src/main.tsx` — entry point
